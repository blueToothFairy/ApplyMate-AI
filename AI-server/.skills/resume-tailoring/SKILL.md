# Resume Tailoring Skill

## Rules

1. Preserve truth from the original CV.
2. Use JD keywords only when supported by evidence in the original CV.
3. Missing JD skills belong in gap analysis, not as direct CV claims.
4. Rewrite bullets using action + scope + technology + result.
5. Prefer concise, recruiter-friendly language.
6. Keep the CV defensible in an interview.

## Bullet Formula

`Action verb + technical scope + implementation detail + outcome`

Example:

- Weak: Worked on backend.
- Strong: Implemented REST API endpoints for workshop registration using Node.js and MongoDB, improving maintainability through modular controller/service structure.

## Safety

If a claim is not supported by the original CV, flag it for user confirmation instead of adding it to the final CV.
