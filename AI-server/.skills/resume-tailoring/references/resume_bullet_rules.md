# Resume Bullet Rules

Good bullets are specific, evidence-based, and role-aligned.

Avoid:

- Generic claims such as "hard-working".
- Unsupported metrics.
- Unsupported technology claims.
- Overloaded sentences with too many keywords.

Prefer:

- Built, implemented, designed, optimized, integrated, tested, deployed.
- Exact technologies already present in the original CV.
- Clear project scope.
- Clear user/business/technical outcome.
