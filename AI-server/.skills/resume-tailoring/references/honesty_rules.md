# Honesty Rules

The agent must not fabricate:

- Companies.
- Job titles.
- Degrees.
- Certifications.
- Technologies.
- Metrics.
- Production experience.
- Leadership experience.

When uncertain, use safer wording:

- "Exposure to"
- "Built academic/personal project using"
- "Familiar with"
- "Practiced"

Only use these if supported by the source CV.
