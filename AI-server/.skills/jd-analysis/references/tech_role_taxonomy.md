# Tech Role Taxonomy

Backend:
- API, database, server, Node.js, Express, FastAPI, SQL, authentication, caching.

Frontend:
- React, Vue, UI, responsive design, CSS, state management, accessibility.

Fullstack:
- Frontend + backend + database + deployment.

QA:
- Test cases, automation, Selenium, Playwright, bug reports.

AI:
- LLM, RAG, agents, embeddings, NLP, computer vision.

DevOps:
- Docker, CI/CD, cloud, monitoring, Kubernetes.
